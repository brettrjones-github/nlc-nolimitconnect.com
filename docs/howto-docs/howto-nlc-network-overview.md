## NLC Network Host Components

An NLC Network Host consists of:

- A **Host Listing Service**
- A **Connection Test Service**
- *(Optional)* One or more of the following:
  - Group
  - ChatRoom
  - Random Connect

---

## NLC Network Host Requirements

To run an NLC Network Host, you will need:

- A **device or VPS** running NLC  
  *(Preferably online 24/7 to ensure network availability for users)*
- An **open network port**
- A **static or fixed IP address**  
  ➡️ [Fixed vs. Static IP Explanation](howto-fixed-vs-static-ip.md)
- *(Optional)* **Domain names**  
  ➡️ [Domain Name Uses](howto-domain-names.md)
