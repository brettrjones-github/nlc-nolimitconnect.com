# How To Host Group, ChatRoom or Random Connect

## To Be A Host you **MUST** have a open network port
➡️ [How To Open Network Port](../port-forwarding.md)

## Enable the hosting plugin with desired permission level
➡️ [How To Set Plugin Permissions](howto-plugin-permission-levels.md)

1. Navigate NLC Settings->Share Permission List page.
2. Configure plugin-specific permission settings.
3. Restart NLC to assure changes take effect.
4. May take up to 15 minutes for your host to become visible in host listings
